Method Details

~ (void) add(keybinding, callback)

Add a keybinding

Examples:

Assign <Super>+G to lunch google-chrome 

var keybindings = modules.keybindings;
var spawn = modules.spawn;
var callback = function() {
  spawn("google-chrome");
};
keybindings.add("<Super>c", callback);

Parameters:

  • keybinding (String) — the keybinding string
  • callback (Function) — the function to be called when keydown

~ (void) apply() Private

Apply keybindings, should be called after all keybindings was added. Config are written using gsettings.
