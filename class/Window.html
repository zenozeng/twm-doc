# (void) getAll()

Get all windows which is shown in current workspace

Examples:

Move every windows in current workspace to (0, 0) and resize to 100*100 

(new Window()).getAll().forEach(function(win){
  win.setArea(0, 0, 100, 100);
});

Note: (0, 0) was at the left-bottom of the top bar

Parameters:

  • x (Number) — position x (px)
  • y (Number) — position y (px)
  • width (Number) — width (px) (border & shadow should not be included)
  • height (Number) — height (px) (border & shadow should not be included)
